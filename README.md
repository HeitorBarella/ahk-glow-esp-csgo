# ahk-glow-esp-csgo


Controls (Numpad):

7 up 4 down for red

8 up 5 down for blue

9 up and 6 down for green

1 up and 3 down for alpha

Del for pulse glow

2 for health glow

make sure num lock is off

Also Added two keys to kill the script they are END and DEL

The HOME key will restart the script
